## Installation

<Steps>
<Step>
### Download
Download the latest `Cntrl_vX.X.X_macOS.dmg` from the [GitHub Releases](https://github.com/azaek/cntrl/releases).
</Step>

<Step>
    ### Install 
    1. Open the DMG file 
    2. Drag **Cntrl** to your **Applications**
    folder
</Step>

<Step>
### Launch
**Important:** After installing, you must manually open Cntrl for it to start.

1. Open **Applications** folder
2. Double-click **Cntrl**
3. The Cntrl icon will appear in your menu bar

<Callout type="info">
On first launch, macOS may show a security warning. Go to **System Settings â†’ Privacy & Security** and click **Open Anyway**.
</Callout>
</Step>
</Steps>

## Launch at Startup

Right-click the menu bar icon and select **Run at Startup**. This creates a LaunchAgent that starts Cntrl when you log in.

## Verify It's Working

Open your browser and go to:

```
http://localhost:9990/api/status
```

You should see `{"status":"ok"}`.

## Differences from Windows

| Feature     | Windows            | macOS                |
| ----------- | ------------------ | -------------------- |
| Installer   | Interactive `.exe` | Drag-and-drop `.dmg` |
| System Tray | Tray icon          | Menu bar icon        |
| Hibernate   | Full hibernate     | Sleep only           |
| Auto-start  | Startup folder     | LaunchAgent          |
